# Compteur de mots en Java

Ce projet Java permet de compter le nombre total de mots contenus dans un fichier texte.

## Fonctionnement

Le programme :

- lit un fichier texte ligne par ligne ;
- sépare les mots grâce aux espaces ;
- compte le nombre de mots de chaque ligne ;
- affiche le nombre total de mots.

## Technologies utilisées

- Java
- Scanner (`java.util.Scanner`)
- Gestion de fichiers (`java.io.File`)
